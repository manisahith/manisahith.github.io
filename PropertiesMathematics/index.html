Properties.



What is a Property?

A property is any characteristic that applies to a given set.

What are the different types of Properties?

There are 6 properties:

Name of the property Meaning of that property
Closure Property Rational numbers that are closed under addition, subtraction and multiplication but not closed in divison.
Commutative Property Only addition and multiplication are commutative but not subtraction and divison.
Associative Property Rational numbers that only follow the associative property for addition and multiplication but not subtraction and division same like the commutative property.
Distributive Property States, if a, b and c are three rational numbers.
Inverse Property Only addition and multiplication
Identity Property addition and multiplication only

Inverse Property

Additive Inverse

Additive inverse is added to the original number to get 0.
=> Additive inverse of 10 is -10, as 10 + (-10) = 0
=> Additive inverse of -9 is 9, as (-9) + 9 = 0

Multiplicative Inverse

Multiplicative inverse is added to the original number to get 1.
=>p/q x q/p = 1
For example: 2/7 x 7/2 = 1

Distributive Property

Distributive property means dividing that or it into parts. Distributive property makes you easy for simplifying difficult problems.
For example: 3 x (1 + 2)= 3 x 1 + 3 x 2

Associative Property

Associative property is a property of some related operations.
For example: 3 + (1 + 2)= (3 + 1) + 2

Commutative Property

Commutative property is telling that order of terms doesn’t matter while performing arithmetic operations.
For Example: 3 + 1 = 1 + 3 or 3 x 1 = 1 x 3
